# 📊 Noise Monitoring and Analysis System

## 📌 Overview
A Python-based application to monitor and analyze environmental noise levels over time.

---

## 🚀 Features
- Collects daily noise data
- Validates user input
- Calculates average noise levels
- Detects threshold violations (85 dB)
- Generates structured data payload
- Visualizes data using graphs

---

## 🛠 Technologies Used
- Python
- Pandas
- Matplotlib

---

## ⚙️ How It Works
1. User inputs location and date range
2. System collects noise levels for each day
3. Calculates statistics
4. Displays graph of noise trends

---

## ▶️ How to Run
pip install -r requirements.txt  
python main.py  

---

## 📊 Output
- Noise level graphs
- Average noise calculation
- Threshold comparison

---

## 📌 Applications
- Environmental monitoring
- Smart city noise analysis
- Industrial noise tracking
